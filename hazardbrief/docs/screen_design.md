## 1. 全体構成

### 1.1 ルーティング

| パス | ビュー | 認証 | 概要 |
|------|--------|------|------|
| `/login` | LoginView | 不要 | Auth0 ログイン |
| `/dashboard` | DashboardView | 必要 | 物件一覧 |
| `/properties/new` | PropertyNewView | 必要 | 物件登録フォーム |
| `/properties/:id` | PropertyDetailView | 必要 | レポート表示 |
| `/settings` | SettingsView | 必要 | 設定 |

### 1.2 技術スタック

- Vue 3 (Composition API + `<script setup>`)
- TypeScript (strict mode)
- Tailwind CSS (ユーティリティファースト)
- Leaflet (地図表示)
- Auth0 (認証)
- Axios (API 通信)

## 2. 共通レイアウト

### 2.1 ヘッダー (AppHeader)

```
┌─────────────────────────────────────────────────────────┐
│  [HB] HazardBrief    物件一覧  物件登録  設定      ユーザー名  ログアウト │
└─────────────────────────────────────────────────────────┘
```

- 固定ヘッダー (白背景、下線ボーダー)
- アクティブリンクは青色ハイライト
- 認証済みの場合はユーザー名とログアウトボタンを表示

### 2.2 カラースキーム (リスクレベル)

| リスクレベル | 色 | Tailwind クラス |
|---|---|---|
| 低 (low) | 緑 | `bg-green-50 border-green-200 text-green-800` |
| 中 (medium) | 琥珀 | `bg-amber-50 border-amber-200 text-amber-800` |
| 高 (high) | オレンジ | `bg-orange-50 border-orange-200 text-orange-800` |
| 要確認 (unknown) | グレー | `bg-gray-50 border-gray-200 text-gray-600` |

> **注意**: 赤 (red) は使用しない。オレンジ〜黄色系でリスクを表現する。

```
┌──────────────────────────────┐

│         [HB] ロゴ            │

## 5. 物件登録 (PropertyNewView)

**パス**: `/properties/new`

```
┌──────────────────────────────────────────────────────────┐
│ AppHeader                                                │
├──────────────────────────────────────────────────────────┤
│ ← 戻る   物件を登録                                        │
│                                                          │
│ ┌──── フォーム ──────────┐ ┌──── 地図プレビュー ──────┐  │
│ │ 住所 *                 │ │                           │  │
│ │ [入力欄]               │ │   [Leaflet 地図]           │  │
│ │ ↓ 自動でプレビュー     │ │                           │  │
│ │                        │ │  緯度: 35.xxxx             │  │
│ │ 物件名 (任意)           │ │  経度: 139.xxxx            │  │
│ │ [入力欄]               │ │                           │  │
│ │                        │ └───────────────────────────┘  │
│ │ メモ (任意)             │                              │
│ │ [テキストエリア]        │                              │
│ │                        │                              │
│ │ [物件を登録してレポートを生成] │                       │
│ └────────────────────────┘                              │
└──────────────────────────────────────────────────────────┘
```

### インタラクション

1. 住所入力 (800ms デバウンス) → 国土地理院 API でジオコーディング → 地図更新
2. 送信 → `POST /api/properties` → 成功時に詳細ページへリダイレクト

```

- レベルバッジ (低/中/高/要確認)
- 説明文 (なぜこのレベルか)
- 対策ヒント (必ず表示)
