陸運業 / 電気機器 / 電気･ガス業 / 非鉄金属 / 食料品

### 前処理の内容

**等加重平均リターン**で計算（時価総額加重ではない）。
各セクター・各日の銘柄リターンを単純平均 → 週次（金曜日基準）に `resample('W-FRI')` → 複利換算（`(1+r).prod()-1`）。

**週次ランク付け**は「その週に最もリターンが高かったセクターが1位」の相対順位。
K-Means のインプット特徴量はこのランク行列（絶対値ではなく相対強弱）。

## 2. 発見されたローテーションパターンの一覧と解釈

K-Means (k=5) が 262週を以下の5状態に分類した。

### 状態一覧

| 状態 | 名称 | 出現週 | 市場平均週次リターン | 性質 |
|---|---|---|---|---|
| **S0** | 電機・非鉄主導/海運安 | 55週 (21%) | **+0.92%** | グロース強気 |
| **S1** | 空運・小売主導/海運安 | 45週 (17%) | **−1.26%** | 調整・防御 |
| **S2** | 海運・水産主導/情通安 | 42週 (16%) | **−0.69%** | 弱い反発・ディフェンシブ |
| **S3** | 鉱業・銀行主導/情通安 | 73週 (28%) | **+1.06%** | バリュー強気（最多） |
| **S4** | 海運・情通主導/医薬安 | 47週 (18%) | **+0.72%** | 海運主導・混合強気 |

### 各状態の詳細解釈

#### S0 — グロース強気フェーズ

- **上位**: 電気機器+2.34%, 非鉄金属+2.11%, 機械+2.03%, 精密機器+1.96%, 輸送用機器+1.94%
- **下位**: 海運業−0.24%, 倉庫−0.23%, 電気ガス−0.05%
- **解釈**: 半導体・製造業主導の典型的リスクオン。海運からの資金流出がバロメータ。円安・米国景気拡大期に対応。

#### S1 — 全面調整フェーズ（唯一の弱気状態）

- **上位**: 空運+0.21%, 小売−0.02%, 医薬品−0.05%（相対的マシ）
- **下位**: 鉄鋼−2.65%, 鉱業−2.63%, 非鉄−2.44%, 海運−5.36%
- **解釈**: 市場平均−1.26%と唯一の弱気状態。コモディティ系が総崩れ。空運・小売が相対的に耐えるディフェンシブ回帰。VIXスパイク期・円高ショック期と一致しやすい。

#### S2 — ディフェンシブ底値圏

- **上位**: 海運+1.48%, 水産+0.34%, 銀行+0.32%, 食料品+0.24%
- **下位**: 情報通信−1.92%, サービス−1.79%, 電気機器−1.71%
- **解釈**: 市場平均−0.69%。海運単独でかろうじてプラスだが全体は弱い。成長株・IT系から資金が逃げ、食料品・銀行という低ベータ銘柄へ。調整局面の中盤〜終盤の通過点。

#### S3 — バリュー強気フェーズ（最優勢状態）

- **上位**: 鉱業+3.25%, 銀行+2.52%, 石油石炭+2.42%, 非鉄+2.33%, 海運+2.10%
- **下位**: 空運+0.17%, サービス−0.18%, 情報通信−0.54%
- **解釈**: 市場平均+1.06%と最高水準。資源・金融が主役の典型的インフレ環境対応。2023年以降の主役で73週と最多出現。日銀利上げ・資源高・中国需要期待と構造的に整合。

#### S4 — 海運主導・混合強気

- **上位**: 海運+4.49%, 情通+1.60%, 鉄鋼+1.42%, サービス+1.41%, 非鉄+1.12%
- **下位**: 水産+0.02%, 食料品+0.01%, 医薬品−0.37%
- **解釈**: 海運が突出して強いが、IT・サービスも同時に上昇する珍しい組み合わせ。輸出拡大×デジタル需要の重なりか。2021〜2022年のコンテナ運賃急騰期が中心。

## 3. 遷移確率の特徴

### 遷移確率行列

```
現在↓ / 次→  S0(電機)  S1(調整)  S2(海運水産)  S3(銀行資源)  S4(海運IT)
S0(電機):       18%      15%       16%         35%          16%
S1(調整):       27%      20%       16%         22%          16%
S2(海運水産):   19%      19%       10%         21%          31%
S3(銀行資源):   19%      17%       22%         32%          10%
S4(海運IT):     23%      17%       11%         26%          23%
```

### よく起きる遷移（確率 30% 超）

| 遷移 | 確率 | サンプル数 | 解釈 |
|---|---|---|---|
| **S0 → S3** | **35%** | n=19 | グロース強気→バリュー強気。電機主導の後、鉱業・銀行に資金シフト |
| **S3 → S3** (自己) | **32%** | n=23 | バリュー強気が最も継続しやすい。最長数週間連続を記録 |
| **S2 → S4** | **31%** | n=13 | ディフェンシブ底値→海運IT混合強気。反発フェーズの始動 |

### 珍しい遷移（確率 10%〜11%）

| 遷移 | 確率 | 解釈 |
|---|---|---|
| **S2 → S2** (自己) | **10%** | ディフェンシブは長続きしない。海運水産主導は通過点 |
| **S3 → S4** | **10%** | バリュー強気→海運IT混合への直行は起きにくい |
| **S4 → S2** | **11%** | 海運IT主導から急にディフェンシブへは少ない |

### 自己遷移確率と期待継続週数

| 状態 | 自己遷移確率 | 理論的期待継続 |
|---|---|---|
| S3 鉱業・銀行主導 | **32%** | **1.5週** |
| S4 海運・情通主導 | **23%** | **1.3週** |
| S0 電機・非鉄主導 | **18%** | **1.2週** |
| S1 空運・小売主導 | **20%** | **1.2週** |
| S2 海運・水産主導 | **10%** | **1.1週** |

**⚠️ 全状態で期待継続が1〜2週と極めて短い。**
直近60週の実データでも1週での状態切り替えが多発しており、「明確なフェーズ」というより「週単位の風向き」に近い構造になっている。

### 構造的観察：S3 はハブ状態

全ての状態から S3 への遷移が 21〜35% あり、市場全体が最終的にバリュー強気に収束する傾向。
S3 からの主要な脱出先は S2（22%）で、軽い修正を経由してから次のサイクルへ移行するパターンが多い。

## 4. 予測モデルの性能と限界

### LightGBM Walk-Forward 結果

| 指標 | 値 |
|---|---|
| モデル精度（Walk-Forward CV） | **15.8%** |
| ランダム予測ベースライン | 20.0%（5クラス均等） |
| 直近予測信頼度 | 99.6%（S3継続予測） |

**⚠️ モデル精度がランダムを下回っている（15.8% < 20%）。**

### 原因分析

**① 状態が週単位で不安定すぎる**
自己遷移確率が最大32%では、前週の状態が翌週を予測する情報量がほぼない。教師ラベル自体がノイジー。

**② Walk-Forward のサンプル不足**
262週を 70/30 分割すると訓練183週・テスト79週。5状態 × 33セクター × ラグ特徴量に対してサンプルが少なく汎化しない。

**③ モデルの過信（99.6%信頼度）**
LightGBM の softmax 出力がほぼ100%に集中するのは過学習の典型。前週が S3 → 「S3が最多状態なので常にS3を予測」で精度が出ているだけの可能性が高い。

**④ マクロ情報の欠如**
リターン・ランクのラグ特徴量だけでは、状態切り替えを引き起こす政策変更・地政学リスクを捉えられない。

## 5. 実運用上の注意点・改善案

### ⚠️ すぐに対処すべき問題

| 問題 | 対処案 |
|---|---|
| 予測精度がランダム以下 | 予測機能は「参考情報」として扱い、投資判断への直接利用を避ける |
| 99.6%信頼度は誤解を招く | UI の信頼度バーに「過去精度15%」を必ず併記する（現状実装済） |

### モデル改善案（中期）

**① 状態数を削減（k=3 程度）**
「強気 / 中立 / 弱気」の3状態にすれば各状態のサンプルが増え、自己遷移確率が高まり予測が意味を持ちやすくなる。

**② 月次集計で予測する**
週次は変動が大きすぎる。月次状態（61ヶ月）で予測すれば、より安定したシグナルが得られる可能性がある。

**③ マクロ特徴量を追加**
`exchange_rates`（ドル円）、`margin_trading_weekly`（PL比率）、`market_pressure_daily`（信用残）はすでに DB にある。これらをモデルに取り込むことで状態変化のドライバーを捉えられる。

**④ HMM の活用**
現状 HMM は出力されているが予測には未使用。GaussianHMM の forward algorithm は確率論的に正しい状態系列を推定でき、K-Means よりローテーションモデルに適している。

### 実際的な活用価値

予測精度は低いが、**ヒートマップと状態タイムラインには実用価値がある**。

- **S3（鉱業・銀行主導）が直近24週でも主役**であることをデータが裏付けており、バリュー・金融・資源セクターへの傾斜が継続的に有効であることを確認できる
- 直近24週（2025-09〜）の実績: 非鉄金属+2.73%, 鉱業+1.94%, 銀行+1.55% がトップ3。情報通信−0.39%, サービス−0.12% が唯一のマイナスで、グロース系のアンダーパフォームが継続中
- **S1（調整フェーズ）への早期警戒指標**: 海運業が急落し始めた場合、S0/S3→S1 への遷移が15〜17%で発生する。海運の急変が最も信頼性の高い先行指標
