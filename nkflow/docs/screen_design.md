## 1. 全体構成

### 1.1 ルーティング

| パス | ビュー | 概要 |
|------|--------|------|
| `/` | OverviewView | 日次サマリ・セクターヒートマップ |
| `/timeseries` | TimeseriesView | 個別銘柄の株価チャート・テーブル |
| `/stock/:code` | StockView | 銘柄詳細 (因果関係・シグナル) |
| `/network` | NetworkView | 資金フロー・信用圧力・ネットワークグラフ |
| `/sector-rotation` | SectorRotationView | セクターローテーション予測・ヒートマップ |
| `/news` | NewsView | ニュース一覧・サマリ |

ルーティング方式: **HTML5 History** (`/path`)

### 1.2 技術スタック

- Vue 3 (Composition API + `<script setup>`)
- TypeScript
- Tailwind CSS (ユーティリティファースト)
- Chart.js + vue-chartjs (チャート描画)
- vis-network (ネットワークグラフ)
- Axios (API 通信)

## 2. 共通レイアウト

### 2.1 ヘッダーナビゲーション

```
┌─────────────────────────────────────────────────────────┐
│  nkflow   [概要] [時系列] [資金フロー] [ローテーション] [ニュース]  │
└─────────────────────────────────────────────────────────┘
```

- 固定ヘッダー (白背景、下線ボーダー)
- アクティブなリンクは青色ハイライト
- レスポンシブ対応 (モバイルでは折り返し)

### 2.2 メインコンテンツ

- 背景: `gray-50`
- 最大幅: `max-w-7xl` (1280px)、中央配置
- パディング: `px-4 py-6`

### 2.3 共通 UI パターン

| パターン | スタイル |
|----------|---------|
| カード | 白背景、`border border-gray-200`、`rounded-lg shadow-sm`、`p-4` |
| ローディング | 中央にテキスト「読み込み中...」 |
| エラー | 赤背景バナー (`bg-red-50 text-red-700`) |
| バッジ | 丸角ピル型 (`rounded-full px-2 py-0.5 text-xs`) |
| ボタン (プライマリ) | `bg-blue-600 text-white rounded-md px-3 py-1.5` |
| ボタン (セカンダリ) | `bg-gray-100 text-gray-700 rounded-md px-3 py-1.5` |
| テーブル | `text-sm`、ヘッダー固定、ホバー行ハイライト |

### 2.4 カラースキーム

| 用途 | 色 |
|------|-----|
| 上昇・ポジティブ | 緑系 (`text-green-600`, `bg-green-*`) |
| 下落・ネガティブ | 赤系 (`text-red-600`, `bg-red-*`) |
| 中立・情報 | 青系 (`text-blue-600`) |
| 警告・注意 | 琥珀系 (`text-amber-600`) |
| 無効・背景 | グレー系 (`text-gray-*`) |

### 3.2 表示要素

| 要素 | データソース | 表示形式 |
|------|-------------|---------|
| 日付 | `summary.date` | YYYY-MM-DD |
| 日経平均終値 | `summary.nikkei_close` | ¥{toLocaleString} |
| 騰落率 | `summary.nikkei_return` | ±X.XX% (色分け) |
| レジーム | `summary.regime` | バッジ (risk_on=緑, risk_off=赤, neutral=琥珀) |
| VIX | `fearIndices.vix.value` | 数値 + 前日比% (低=緑、高=赤) |
| Fear&Greed | `fearIndices.btc_fear_greed` | スコア + ラベル |
| 日経平均ミニチャート | `summaryHistory[].nikkei_close` | NikkeiMiniChart (SVG エリアチャート、5日分) |
| 値上がり上位 | `summary.top_gainers` | テーブル (コード・名前・騰落率) |
| 値下がり上位 | `summary.top_losers` | テーブル (コード・名前・騰落率) |
| 年初来高値圏 | `/api/ytd-highs` | テーブル (コード・名前・乖離%) |
| セクターヒートマップ | `summary.sector_rotation` | HeatMap コンポーネント |
| 業種トレンド棒グラフ | `summary.sector_rotation` | SectorTrendBar (バイディレクショナル棒グラフ、上昇/下降カウント付き) |
| 米国セクター | `/api/us-sectors/performance` | 期間切替付き棒グラフ (1D/1W/1M/3M) |

### 3.3 インタラクション

- 銘柄コードをクリック → `/stock/:code` に遷移
- 米国セクター期間ボタンクリック → 期間切替 + 再取得

## 4. 時系列ページ (TimeseriesView)

**パス**: `/timeseries`
**API**: `GET /api/prices/{code}?from={from}&to={to}`

### 4.1 画面構成

```
┌─────────────────────────────────────────────────────┐
│  時系列                                               │
│ ┌─────────────────────────────────────────┐          │
│ │ 銘柄コード: [6758    ] [Enter で検索]    │          │
│ └─────────────────────────────────────────┘          │
│                                                     │
│  期間: [1M] [3M] [6M] [1Y]                          │
│                                                     │
│ ┌─────────────────────────────────────────────────┐  │
│ │              株価チャート (PriceChart)            │  │
│ │         ~~~~~~~~~~~~~~~~~~~~~~~~~~~              │  │
│ │        /                           \             │  │
│ │   ~~~~                              ~~~~         │  │
│ └─────────────────────────────────────────────────┘  │
│                                                     │
│ ┌─────────────────────────────────────────────────┐  │
│ │ 日付       始値     高値     安値     終値  騰落率│  │
│ │ 2026-03-04 15,200  15,350  15,100  15,280 +0.5% │  │
│ │ 2026-03-03 15,150  15,250  15,050  15,200 -0.2% │  │
│ │ ...                                              │  │
│ └─────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────┘
```

### 4.2 表示要素

| 要素 | データソース | 表示形式 |
|------|-------------|---------|
| 銘柄コード入力 | ユーザー入力 | テキストフィールド + Enter トリガー |
| 期間ボタン | 状態 `activePeriod` | 1M(20日)/3M(60日)/6M(120日)/1Y(250日) |
| 株価チャート | `prices[]` | PriceChart (折れ線グラフ、青色、塗りつぶし) |
| 価格テーブル | `prices[]` | 直近30件、日付降順、騰落率色分け |

### 4.3 インタラクション

- 銘柄コード入力 + Enter → API 呼び出し → チャート・テーブル更新
- 期間ボタンクリック → `from` パラメータ変更 → 再取得

## 5. 銘柄詳細ページ (StockView)

**パス**: `/stock/:code`
**API**: `GET /api/stock/{code}`, `GET /api/prices/{code}`

### 5.1 画面構成

```
┌─────────────────────────────────────────────────────┐
│  ← 戻る   6758 ソニーグループ   [電気機器]           │
│                                                     │
│ ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐   │
│ │ 終値    │ │ 騰落率  │ │ 値幅    │ │ 出来高  │   │
│ │ ¥15,280 │ │ +0.53% │ │ 250    │ │ 3.2M   │   │
│ └─────────┘ └─────────┘ └─────────┘ └─────────┘   │
│                                                     │
│  期間: [1M] [3M] [6M] [1Y]                          │
│ ┌─────────────────────────────────────────────────┐  │
│ │              株価チャート (PriceChart)            │  │
│ └─────────────────────────────────────────────────┘  │
│                                                     │
│ ┌── この銘柄が影響を与える ──┐ ┌── 影響を受ける ──┐  │
│ │ 7203 (ラグ 3日, p=0.02)   │ │ 8306 (ラグ 2日)  │  │
│ └───────────────────────────┘ └──────────────────┘  │
│                                                     │
│ ┌── 高相関銘柄 ────────────┐ ┌── クラスタメンバー ─┐ │
│ │ 6501 (r=0.85)           │ │ 6758, 6501, 6702   │ │
│ └─────────────────────────┘ └────────────────────┘ │
│                                                     │
│ ┌── 関連シグナル ──────────────────────────────────┐ │
│ │ 2026-03-04 causality_chain 買い 信頼度 0.78     │ │
│ └─────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────┘
```

### 5.2 表示要素

| 要素 | データソース | 表示形式 |
|------|-------------|---------|
| ヘッダー | `detail.code`, `detail.name`, `detail.sector` | 戻るボタン付き |
| 最新価格カード (4列) | `detail.recent_prices[0]` | 終値・騰落率・値幅・出来高 |
| 株価チャート | `prices[]` | PriceChart (期間切替付き) |
| 因果関係 (causes) | `detail.causes[]` | コード・ラグ日数・p値 |
| 因果関係 (caused_by) | `detail.caused_by[]` | コード・ラグ日数・p値 |
| 高相関銘柄 | `detail.correlated[]` | コード・相関係数 |
| クラスタメンバー | `detail.cluster` | メンバー一覧 |
| 関連シグナル | `detail.signals[]` | 日付・タイプ・方向・信頼度 |

### 5.3 インタラクション

- 戻るボタン → ブラウザバック
- 銘柄コードリンク → `/stock/:code` に遷移
- 期間ボタン → チャート再描画
- シグナル方向: bullish=緑バッジ、bearish=赤バッジ

| サンキーダイアグラム | `FundFlowSankey` | `networkData.edges` |
| ネットワークグラフ | `GraphView` | `networkData` (nodes + edges) |

### 6.3 フィルタモード

| モード | 操作 | パラメータ |
|--------|------|-----------|
| 期間指定 | プリセットボタン (20日/60日/120日) | `period` |
| 範囲指定 | プリセット (5営業日/1ヶ月/3ヶ月) | `date_from`, `date_to` |
| 日付指定 | 日付ピッカー (today/yesterday/先週金曜) | `date_from = date_to` |

### 6.4 アンカーモード

資金フロータイムラインの棒をクリックすると「アンカーモード」に切り替わる:
- 選択した日を基準日としてヘッダーに表示
- 累積スプレッドの折れ線グラフまたはセクター別積み上げ棒グラフに切替
- 「クリア」ボタンで通常モードに復帰
